JavaScript 随机打乱数组 - 洗牌算法

122014年01月

在学习排序算法的时候，经常要用到随机数组，于是就写了一个生成随机数组的方法。算法来自网络，只是修改成了 JavaScript 版本。

基本原理是洗牌算法，首先从所有元素中随机选取一个与第一个元素进行交换，然后在第二个之后选择一个元素与第二个交换，直到最后一个元素。这样能确保每个元素在每个位置的概率都是1/n。
